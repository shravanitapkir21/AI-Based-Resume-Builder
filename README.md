# CareerCraft – Resume Builder
CareerCraft is a responsive resume builder web application
that helps users create clean and ATS-friendly resumes
using professionally designed templates.

## Why this project?
I wanted to build a real product instead of a typical mini project.

## Features
- Multiple resume templates
- Clean and simple UI
- Fully responsive design
- Beginner-friendly project structure

## Tech Stack
- HTML5
- CSS3
- Responsive Web Design

## Application Pages
- Personal Info Page – Collects user basic details
- Education Page – Academic qualifications
- Experience Page – Work / internship details
- Skills Page – Technical and soft skills
- Projects Page – Academic and personal projects
- Preview Page – Final resume preview

## How to Use
1. Open index.html in any browser
2. Fill in personal and education details
3. Select a resume template
4. Preview the resume instantly

## Future Scope
- JavaScript form validation
- Dynamic template switching
- PDF download feature
